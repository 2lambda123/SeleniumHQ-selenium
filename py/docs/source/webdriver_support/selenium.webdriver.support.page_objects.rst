selenium.webdriver.support.page_objects
=======================================

The Page Object Pattern is a useful design pattern to consider when building automated test suites using selenium.

A more comprehensive overview of the pattern can be found here_.

.. _here: http://www.seleniumhq.org/docs/06_test_design_considerations.jsp

***********
Page Object
***********

The base object that all pages must be based on. You implement the `Find By`_ and `Find All By`_ object on this as properties.
